# Public Assets Folder

Place your logo image file here with the name `logo.png`.

The logo should be:
- Format: PNG (or JPG/SVG)
- Recommended size: At least 200px height for best quality
- The logo will be displayed in the navigation header and footer

Supported formats: `.png`, `.jpg`, `.jpeg`, `.svg`

If you have a different filename, update the `src` path in:
- `src/components/Navigation.tsx`
- `src/components/Footer.tsx`
